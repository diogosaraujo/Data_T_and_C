# Data_T_and_C
Data to adjust T&amp;C Model to use for AGU 2025.

LMA: Leaf Mass Per Area -> g cm-2 (Hinojo-Hinojo, in preparation)
Climate Data from ERA5-Land
Ws: Wind Speed -> m/s
Ta: 2m Temperature -> K
Rsw: Surface Solar radiation Downwards -> J m-2
ev: total evaporation -> m of water equivalent
Pr: total precipitation -> m
lat: latitude
lon: longitude
